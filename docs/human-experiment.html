Human Experiment¶

In this page we present the settings, implementation and results of the image classification experiments with human participants that we conducted. We followed the same experiment procedure as Geirhos et al~cite{Geirhos2018GeneralisationIH}. To make sure that the participants understand the task, we included qualification tests and sanity checks. We only considered results from those that passed both. In this experiment, we used images from the ILSRVR 2012 training dataset~cite{ILSVRC2012} and considered a binary classification problem of recognizing car instances. To differentiate between car and non-car instances, we adopted the class hierarchy provided by the ILSVRC2012 dataset. In our experiment, we selected one transformation for each CV-Hazop entry, which corresponds to 8 safety-related transformations: RGB, contrast, defocus blur, brightness, frost, color jitter, jpeg compression, and gaussian noise.

8 transformations included in our experiments¶

For each of the 8 transformations, we sampled uniformly 1000 transformation magnitudes and measured the performance of 5 human participants per each transformation level. To achieve this, we divided the 1000 magnitudes * 8 transformations into batches of 20 images. Each batch is shown 5 times to different participants using the platform Amazon Mechanical Turk. As a result, for image classification, we collected 40000 human performance results.
